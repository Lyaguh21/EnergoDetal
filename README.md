# EnergoDetal

Коммерческий проект реализующий сайт для завода

## ✨ Функционал

- Галерея для просмотра фотографий
- Каталог для просмотра информации и 3d моделей
- Адаптивный дизайн и другое

## 🛠 Cтек

- **Frontend**

  - React + TypeScript
  - MantineUI
  - Zustand
  - Motion
  - React Router
  - ReactThreeFriber

- **Backend**
  - NestJS
  - Postgres
  - Docker
